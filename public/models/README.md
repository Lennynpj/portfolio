# Modèles 3D

Déposer ici les têtes 3D des fondateurs (fournies plus tard) :

- `lenny.glb`
- `gabriel.glb`

**Contraintes d'export :** centré sur l'origine, Y-up, échelle ~1–2 unités,
matériaux PBR, compression Draco/meshopt, < 3 Mo.

Tant que les fichiers ne sont pas présents, le hero affiche un placeholder
porcelaine (voir `src/sections/Hero.tsx`).
